## For development
1. Go to frontend folder
2. Rename .env.example to .env, and fill in the variable
3. run `npm run build`
4. Go to root folder
5. Rename .env.example to .env, and fill in the variable
6. run `npm run start:dev`
7. open your web browser, localhost:${PORT}
